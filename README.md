Flashcard Reviewer
Most flashcard apps have no motivation loop. You open Anki, study a few cards, and stop returning after a week because there is nothing pulling you back.
This app implements the SM-2 spaced repetition algorithm with streaks, XP, and level-ups to build a consistent daily study habit.

Features

SM-2 algorithm schedules cards based on how well you recalled them
Streak counter and XP system to reward daily use
Folder organization for different subjects
Export, import, and shuffle cards
Fully offline, runs in any browser

How to run

Download FlashcardReviewer_v1.10.html
Open it in any browser
No installation needed

Tech used
HTML, CSS, JavaScript
